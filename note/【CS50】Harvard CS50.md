# Harvard CS50

[Harvard CS50](https://cs50.harvard.edu/x/)是哈佛大学一套非常有意思的 CS 入门课，适合新手学习和了解 CS 基础知识，大佬可以开倍速听一听，就当锻炼英语听力了。

## Week 0 Scratch

这一章介绍了计算机科学中的一些基本概念，比如二进制、编码、算法、人工智能等，后半段主要讲[Scratch](https://zh.wikipedia.org/zh-cn/Scratch)，如果已经有一定的编程语言经验可以跳过这段。

## Week 1 C

这一章以C语言为起点，介绍了编程中的基本概念，比如变量、函数、条件语句、循环语句等，包含若个有趣的小案例，整个编程基础知识部分不超过三个小时。如果你第一次接触编程，可以跟着课程边学边写，比看其他文档或教程高效得多。
